# Neon Drum Machine

Client-side drum machine. Next.js + Tailwind. Web Audio API synthesizes every sound, no audio files, no backend.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Features

- 4x4 grid of 16 colorful neon pads (kick, snare, hats, clap, toms, cowbell, rim, crash, ride, shaker, bass, zap, laser)
- Click a pad: plays the sound + ripple/glow animation
- 16-track step sequencer, 8 beats per bar
- Play/Stop, BPM slider (60-200), Clear
